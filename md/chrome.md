# Open all recently closed tabs

```bash
    ctrl + shift + t
```

# Open new tab

```bash
    ctrl + t
```

# Open new window

```bash
    ctrl + n
```

# Open new window in incognito mode

```bash
    ctrl + shift + n
```

# Open developer tools

```bash
    ctrl + shift + i
```

# Bookmark any tab

```bash
    ctrl + d
```

# Go to url bar

```bash
    ctrl + l
```

# Go to search bar

```bash
    /
```

# Close current tab

```bash
    ctrl + w
```

# Switch between tabs

```bash
    ctrl + (pg up or pg dn)
```

# Go to previous or forwarded page

```
    alt + (left arrow or right arrow)
```

# Open link in new tab

```
    ctrl + left click
```
